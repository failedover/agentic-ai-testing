# File Organization Categories
# This file defines the categories used to organize files.
# Edit the category names and descriptions to suit your needs.
# The AI will use these definitions to decide where each file belongs.
# FORMAT: Each category starts with "## CategoryName" followed by a description.

## Finances
Documents related to budgets, invoices, receipts, expenses, taxes, bank statements,
payroll, billing, quotes, payments, accounting, subscriptions, and financial reports.

## Work Projects
Files related to professional work, client projects, deliverables, proposals,
presentations, meeting notes, project plans, timelines, contracts, and work reports.

## Personal
Personal documents such as letters, journals, health records, insurance documents,
ID scans, travel plans, family documents, and anything related to personal life.

## Photos & Media
Images, videos, audio files, recordings, screenshots, artwork, graphics,
design assets, wallpapers, and multimedia content.

## Learning & Research
Study materials, course notes, tutorials, research papers, articles, reference guides,
textbooks, e-books, certifications, and any educational content.

## Software & Code
Source code files, scripts, configuration files, development projects, documentation
for software, API references, database files, and technical specs.

## Archives & Backups
Compressed files, backups, exports, old versions, zip archives, and any file
that appears to be a backup or archived copy of something.

## Miscellaneous
Files that do not clearly fit into any other category. Used as a fallback.
